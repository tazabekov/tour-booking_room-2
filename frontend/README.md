# TravelHub - Платформа бронирования туров

Современный веб-сайт для бронирования туров, созданный с использованием React, TypeScript, Vite и Tailwind CSS.

## Особенности

- **Главная страница**: Hero-секция с поиском, популярные туры, направления
- **Каталог туров**: Фильтрация по стране и ценовому диапазону, поиск
- **Страница тура**: Галерея изображений, детальное описание, программа тура, что включено/не включено
- **Страница бронирования**: Форма бронирования с валидацией, подтверждение через модальное окно
- **Адаптивный дизайн**: Полностью отзывчивый дизайн для всех устройств
- **Современный UI**: Чистый дизайн с Tailwind CSS, плавные анимации, красивые карточки

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - Типизация
- **Vite** - Сборщик и dev-сервер
- **React Router DOM** - Маршрутизация
- **Tailwind CSS** - CSS фреймворк
- **Inter Font** - Современный шрифт от Google Fonts

## Установка и запуск

### Предварительные требования

- Node.js 18.x или выше
- npm или yarn

### Установка

1. Перейдите в директорию проекта:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Предварительный просмотр продакшен сборки

```bash
npm run preview
```

## Маршруты

- `/` - Главная страница
- `/tours` - Каталог всех туров
- `/tours/:id` - Детальная страница конкретного тура
- `/booking/:tourId` - Страница бронирования тура

## Функциональность

### Фильтрация
- Фильтр по странам (Турция, Египет, ОАЭ, Таиланд, Мальдивы, Италия)
- Ценовой диапазон с двумя слайдерами (минимум и максимум)
- Текстовый поиск по названию, стране, городу и описанию тура

### Моковые данные
Приложение содержит 10 туров в следующие страны:
- Турция (Анталия, Бодрум)
- Египет (Хургада, Шарм-эль-Шейх)
- ОАЭ (Дубай, Абу-Даби)
- Таиланд (Бангкок/Пхукет, Паттайя)
- Мальдивы (Мале)
- Италия (Рим/Венеция)

Каждый тур включает:
- ID, название, описание
- Страна и город
- Цена и продолжительность
- Рейтинг
- Массив изображений (3 штуки с Unsplash)
- Программа по дням
- Список включенных и исключенных услуг
- Особенности тура

### Бронирование
Форма бронирования включает:
- Имя (обязательно)
- Email (обязательно, с валидацией)
- Телефон (обязательно)
- Количество туристов (1-10)
- Дата начала тура (не раньше сегодня)

После отправки показывается модальное окно с подтверждением всех данных.

## Дизайн

Приложение использует современный дизайн 2025 года:
- **Цвета**:
  - Primary: #14B8A6 (teal)
  - Accent: #F97316 (coral orange)
  - Background: #FAFAFA (light gray)
- **Шрифт**: Inter (Google Fonts)
- **Стиль**: Минималистичный, с большим количеством whitespace
- **Карточки**: Закругленные углы (rounded-2xl), мягкие тени
- **Анимации**: Плавные переходы на hover, scale эффекты
- **Адаптивность**: Mobile-first подход
